### rquest_dto
The classes used by the rest of the core are defined in here.
They all inherit from [`BaseDto`](/resolve/core_api_ref/rquest_dto/base_dto).

| component | description |
|:----------|------------:|
| [`activity_job`](/resolve/core_api_ref/rquest_dto/activity_job)| A job describing source and payload |
| [`base_dto`](/resolve/core_api_ref/rquest_dto/base_dto)| The base class for other `rquest_dto` classes|
| [`cohort`](/resolve/core_api_ref/rquest_dto/cohort)| A cohort of patients |
| [`file`](/resolve/core_api_ref/rquest_dto/file)| An object for the files generated by a query |
| [`group`](/resolve/core_api_ref/rquest_dto/group)| Groups of patients within a cohort |
| [`query`](/resolve/core_api_ref/rquest_dto/query)| The different types of query made against the database |
| [`result`](/resolve/core_api_ref/rquest_dto/result)| The results of an RQuest query |
| [`rule`](/resolve/core_api_ref/rquest_dto/rule)| Generating a rule from an RO-crate |
