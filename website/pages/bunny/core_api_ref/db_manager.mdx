### `SyncDBManager(BaseDBManager)`
```python
class SyncDBManager(
  username: str,
  password: str,
  host: str,
  port: int,
  database: str,
  drivername: str,
  schema: Optional[str] = None,
)
```

Class for managing a database connection through a standard database engine.
Uses the parameters to build a connection URL.
