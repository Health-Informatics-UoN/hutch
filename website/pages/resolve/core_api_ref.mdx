## Core API reference
The scripts in `core` are called by the Resolve daemon and command-line interface (CLI).

### rquest-dto
The classes used by the core functions are held in [rquest-dto](/resolve/core_api_ref/rquest-dto).

|component|description|
|:---|---:|
|[`constants`](/resolve/core_api_ref/constants)| provides names for [`File`s](/resolve/core_api_ref/rquest_dto/file)|
|[`db_manager`](/resolve/core_api_ref/db_manager)| manages database connections for resolve |
|[`entities`](/resolve/core_api_ref/entities)| Object Relational Mappings for the OMOP-CDM |
|[`enums`](/resolve/core_api_ref/enums)| Resolve enumerations |
|[`execute_query`](/resolve/core_api_ref/execute_query)| executes queries against the database |
|[`logger`](/resolve/core_api_ref/logger)| logs Resolve's activities |
|[`obfuscation`](/resolve/core_api_ref/obfuscation)| Result obfuscation to protect personally identifiable information |
|[`parser`](/resolve/core_api_ref/parser)| parses command-line arguments for resolve-cli |
|[`query_solvers`](/resolve/core_api_ref/query_solvers)| Solves different types of query against the database |
|[`results_modifiers`](/resolve/core_api_ref/results_modifiers)| configures [obfuscation](/resolve/core_api_ref/obfuscation) |
|[`setting_database`](/resolve/core_api_ref/setting_database)| builds [`db_manager`](/resolve/core_api_ref/db_manager) for resolve |
|[`settings`](/resolve/core_api_ref/settings)| Provides settings for other core modules by reading environment variables |
|[`task_api_client`](/resolve/core_api_ref/task_api_client)| A client for interacting with the Task API |
